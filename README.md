# 智能教学助手

## 系统概述

这是一个基于RAG（检索增强生成）技术的智能教学助手系统，集成了知识库管理、教学内容设计、考核内容生成和考试系统功能。系统分为教师端和学生端，提供完整的教学辅助功能。

## 系统架构

### 后端架构
- **FastAPI**: 现代化的Python Web框架
- **SQLAlchemy**: 数据库ORM
- **SQLite**: 轻量级数据库
- **JWT**: 用户认证
- **LangChain**: RAG核心框架
- **ChromaDB**: 向量数据库

### 前端架构
- **React**: 用户界面框架
- **Ant Design**: UI组件库
- **Axios**: HTTP客户端

## 功能模块

### 教师端功能

#### 1. 知识库管理
- **多文件上传**: 支持PDF、Word文档批量上传
- **文档处理**: 自动解析文档内容并向量化存储
- **文件管理**: 查看、删除知识库文件
- **OCR支持**: 自动处理扫描版PDF文档

#### 2. 知识库问答
- **智能问答**: 基于知识库内容的智能问答
- **实时响应**: 快速准确的答案生成

#### 3. 教学内容设计
- **课程大纲输入**: 输入课程大纲和要求
- **AI设计**: 自动生成详细的教学内容设计
- **结构化输出**: 包含教学目标、教学步骤、评估方法等

#### 4. 考核内容生成
- **多学科支持**: 支持通用学科、计算机类、编程类等
- **题型丰富**: 包含概念题、填空题、简答题、编程题
- **智能生成**: 基于课程大纲自动生成考核内容
- **分值设置**: 教师可设置每道题的分值

#### 5. 考试管理
- **考试创建**: 基于生成的考核内容创建考试
- **考试列表**: 查看已创建的考试
- **学生统计**: 查看参与考试的学生数量

### 学生端功能

#### 1. 知识库问答
- **智能问答**: 向知识库提问获取答案
- **学习辅助**: 帮助学生理解课程内容

#### 2. 考试系统
- **考试列表**: 查看可参加的考试
- **在线答题**: 支持多种题型的在线答题
- **计时功能**: 考试时间限制和倒计时
- **自动评分**: 客观题自动评分
- **成绩查看**: 查看考试成绩

## 技术特点

### RAG技术
- **文档检索**: 基于语义相似度的文档检索
- **内容生成**: 结合检索结果和LLM生成答案
- **知识更新**: 支持知识库的动态更新

### 用户体验
- **响应式设计**: 适配不同屏幕尺寸
- **实时反馈**: 操作结果实时反馈
- **进度提示**: 长时间操作进度提示

### 安全性
- **JWT认证**: 安全的用户认证机制
- **权限控制**: 基于角色的权限管理
- **数据验证**: 输入数据严格验证

## 安装和运行

### 环境要求
- Python 3.8+
- Node.js 14+
- npm 或 yarn

### 后端安装
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 前端安装
```bash
cd frontend
npm install
npm start
```

### 数据库初始化
系统首次运行时会自动创建数据库表结构。

## 使用说明

### 教师使用流程
1. **注册/登录**: 以教师身份注册并登录系统
2. **上传文档**: 上传教学相关的PDF或Word文档
3. **知识库问答**: 使用知识库进行问答测试
4. **设计教学内容**: 输入课程大纲，生成教学内容
5. **生成考核内容**: 基于课程大纲生成考核题目
6. **创建考试**: 选择题目并设置分值，创建考试

### 学生使用流程
1. **注册/登录**: 以学生身份注册并登录系统
2. **知识库问答**: 向知识库提问学习
3. **参加考试**: 查看可参加的考试并开始答题
4. **查看成绩**: 查看考试成绩和答题详情
